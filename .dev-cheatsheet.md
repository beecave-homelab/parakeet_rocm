## Start Development Environment

```bash
# First time setup
docker compose -f docker-compose.dev.yaml build

# Start dev server (hot-reload enabled)
docker compose -f docker-compose.dev.yaml up

# WebUI: http://localhost:7861
```

## Hot-Reload Workflow

```bash
# 1. Edit code in IDE
vim parakeet_rocm/webui/app.py

# 2. Save file (Ctrl+S)

# 3. Restart container (< 5 seconds)
docker compose -f docker-compose.dev.yaml restart

# 4. Refresh browser â†’ Changes live!
```

## Common Commands

```bash
# Stop dev environment
docker compose -f docker-compose.dev.yaml down

# View logs
docker compose -f docker-compose.dev.yaml logs -f

# Run tests inside container
docker compose -f docker-compose.dev.yaml exec parakeet-rocm-dev pytest tests/unit/ -v

# Interactive shell
docker compose -f docker-compose.dev.yaml exec parakeet-rocm-dev bash

# Run CLI inside container
docker compose -f docker-compose.dev.yaml exec parakeet-rocm-dev \
  parakeet-rocm transcribe /data/sample.wav
```

## Troubleshooting

```bash
# Changes not reflecting? Restart container
docker compose -f docker-compose.dev.yaml restart

# Still not working? Force rebuild
docker compose -f docker-compose.dev.yaml build --no-cache
docker compose -f docker-compose.dev.yaml up

# Check mounted files
docker compose -f docker-compose.dev.yaml exec parakeet-rocm-dev \
  ls -la /app/parakeet_rocm/
```

## Production vs Development

| Aspect | Production | Development |
|--------|-----------|-------------|
| Source code | Copied into image | Mounted (hot-reload) |
| Rebuild needed | Yes (every change) | No (only deps) |
| Iteration speed | 2-5 min | 5-10 sec |
| Command | `docker compose up` | `docker compose -f docker-compose.dev.yaml up` |
